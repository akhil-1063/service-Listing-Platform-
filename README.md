# Trial Task - Service Listing Platform

## Objective

Develop a basic web application that allows users to browse and search service providers.

## Technology Stack

### Frontend
- Next.js (Preferred) / React.js
- Tailwind CSS (Preferred)

### Backend
- Node.js
- Express.js

### Database
- MongoDB

## Features

### Frontend

1. Responsive Landing Page

Include:
- Hero Section
- About Section
- Featured Services Section
- Contact Section

2. Service Listing Page

Display:
- Service Name
- Location
- Category
- Description
- Image

3. Search Functionality

Users should be able to search by:
- Service Name

4. Filter Functionality

Users should be able to filter by:
- Category
- Location

---

### Backend

Create REST APIs for:

#### Service Provider

- Create Service Provider
- Get All Service Providers
- Get Single Service Provider
- Update Service Provider
- Delete Service Provider

---

### Database

Store:

- Name
- Category
- Location
- Description
- Image URL

---

- Source code pushed to GitHub
- Clean commit history
- README with setup instructions
- Working application
